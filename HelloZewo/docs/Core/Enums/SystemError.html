SystemError

public enum SystemError : Error

POSIX error representation.

Example:

func listen(socket: Int32, backlog: Int32) throws {
    let result = listen(socket, backlog)

    guard result != -1 else {
       throw SystemError.lastOperationError
    }
}

do {
    try listen(socket: socket)
} catch SystemError.addressAlreadyInUse {
    print("Another socket is already listening on the same port.")
}
  • lastOperationError

    Error from the last executed operation, if any. The error is created from the value present in errno.

    Warning

    lastOperationError is significant only when the return value of the last operation indicated an error (i.e., -1 from most system calls; -1 or nil from most library functions).
