UnsafeRawBufferPointer

struct UnsafeRawBufferPointer : Collection, RandomAccessCollection
  • bufferSize

    Invokes the given closure on the contents represented as a buffer.

    The withUnsafeBytes(body:) method ensures that the buffer’s lifetime extends through the execution of body. The buffer argument to body is only valid for the lifetime of the closure. Do not escape it from the closure for later use.

    Throws

    Errors thrown from the body closure, if any.

    Swift

    public var bufferSize: Int

    Return Value

    The return value of the body closure, if any.

    A closure that takes a buffer as its sole argument. If the closure has a return value, it is used as the return value of the withUnsafeBytes(body:) method. The buffer argument is valid only for the duration of the closure’s execution.
